Geek Squad Website Build

Description


Data Collector/ Evaluator

As part of a five‑person team, I helped design and test a mockup Geek Squad customer support website. My focus was on usability testing and data collection, where I applied the RITE method to measure efficiency and identify pain points. By analyzing survey data and user feedback, I contributed to meaningful design improvements that enhanced navigation, clarified service descriptions, and streamlined the store locator.

Key Contributions

  • Designed and prototyped a mockup website using Wix Studio
  • Conducted RITE usability testing (task completion, success rate, errors)
  • Collected and analyzed survey data via Google Forms
  • Implemented improvements based on findings
  • Collaborated in a 5‑person team, integrating feedback into prototypes
